Recreating a card game

to demo Figma

Figma demo
design exercise

SET is one of my favorite pattern recognition games. The deck consists of 81 unique cards that vary in four features across three possibilities for each kind of feature: number of shapes, shape, shading, and color.  

The game's setup reminded me of Figma's core competencies essential to building a working design system: variables and components. I got the idea of creating a demo to teach my colleagues, who were new to the tool at the time, how Figma worked, using a game we have played together as a team.

The screen recording above is a quick recreation of the approach I took to layout the full set of cards in under 10 minutes.
